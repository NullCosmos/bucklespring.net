# bucklespring&#46;net

C# .NET port of the original bucklespring source available from: [bucklespring](https://github.com/zevv/bucklespring)

### Prerequisites

Requires OpenAL redistributable to be installed.
[OpenAL Redist](http://openal.org/downloads/)
